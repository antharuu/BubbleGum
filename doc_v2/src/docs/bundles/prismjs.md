## Prism.js

Prism.js is a code hilighter, it is the most used and complete. 

You can find its documentation here: [prismjs.com](https://prismjs.com/index.html)

```html
<!-- Classic usage -->
<pre class="language-css">
    <code>
#my-super-code {
    width: 100%;
    outline: 5px solid #212121;
    border-radius: 15px;
}
    </code>
</pre>


<!-- Bubblegum usage -->
<pre _code="css">
#my-super-code {
    width: 100%;
    outline: 5px solid #212121;
    border-radius: 15px;
}
</pre>
```

Render:

<pre _code="css">
#my-super-code {
    width: 100%;
    outline: 5px solid #212121;
    border-radius: 15px;
}
</pre>

Bubblegum makes prism use as easy as possible
